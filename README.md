jira-vagrant-install
===================

A project that uses Vagrant and Puppet to download and install a copy of JIRA 5.2.10.  This project is very closely based off of Nicola Paolucci's Stash provisioning example https://bitbucket.org/durdn/stash-vagrant-install.git

Check out https://blogs.atlassian.com/2013/03/instant-java-provisioning-with-vagrant-and-puppet-stash-one-click-install/ for more details
